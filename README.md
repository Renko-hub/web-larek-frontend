- ### Документация проекта
-
- #### Описание проекта
-
- Проект представляет собой веб-приложение интернет-магазина, построенное на современном стеке технологий с использованием классов и методов ES6+.
-
- **Цель**: Предоставление пользователям функционала для просмотра товаров, добавления их в корзину, оформления заказов и получения уведомлений о результатах покупок.
-
- **Основные возможности**:
- - Просмотр каталога товаров.
- - Добавление товаров в корзину.
- - Оформление заказов с выбором способа оплаты.
- - Отправка контактных данных.
- - Оповещение пользователей о завершении процесса покупки.
-
- ***
-
- #### Структура базового кода
-
- Проект организован следующим образом:
-
- - `index.ts`: Главная точка входа приложения, управляющая инициализацией процессов загрузки и рендеринга.
- - `Page.ts`: Основной компонент, формирующий интерфейс и реагирующий на пользовательские действия.
- - `Model.ts`: Модель данных, включая описание структуры товаров и корзины.
- - `BaseModal.ts`: Базовый класс для реализации любых модальных окон.
- - `Forms.ts`: Библиотека функций для валидации и обработки форм.
- - `Components.ts`: Набор компонентов для удобства работы с элементами пользовательского интерфейса.
- - Отдельные файлы для каждой функциональности модального окна (`BasketModal`, `CardModal`, `OrderModal`, `ContactsModal`, `SuccessModal`), каждый из которых реализует свою уникальную логику.
-
- В проекте также предусмотрены вспомогательные утилиты для работы с DOM, такие как `cloneTemplate`, `ensureElement`, `bem`, обеспечивающие безопасность и эффективность взаимодействия с разметкой и стилями.
-
- ***
-
- #### Типы данных, с которыми работает приложение
-
- Основой приложения являются объекты типа `Product` расположен в '../types/index', описывающие структуру товаров:
-
- - `id`: Уникальный идентификатор товара.
- - `title`: Название товара.
- - `category`: Категория товара.
- - `image`: Путь к изображению товара.
- - `price`: Стоимость товара.
- - `description`: Описание товара (опционально).
- - `quantity`: Количество товара в корзине (опционально).
-
- Эта информация позволяет полноценно функционировать приложению, начиная от отображения каталога и заканчивая обработкой заказов.
-

-
- #### Архитектурные паттерны
-
- Архитектура приложения основана на концепциях MV\*(MVC/MVP/MVVM):
-
- - **View (Представление)** — представлена HTML-разметкой и компонентами интерфейса (карточки товаров, модальные окна, формы).
- - **Controller (Контроллер)** — реализуется в основном файле `Page.ts`, координатор действий и реакций на пользовательские события.
- - **Model (Модель данных)** — содержится в файле `Model.ts`, описывает структуру данных и операции над ними (товары, корзина).
-
- Взаимодействие компонентов осуществляется посредством системы событий, реализованной через класс `EventEmitter`.

- #### Описание функций и методов каждого модуля
-
- 1.  **index.ts**
- Главная точка входа приложения, инициирует начальную загрузку и настройки.
-
- **Ключевые функции:**
- - `window.onload()`: Инициализирует загрузку и рендеринг товаров после полной загрузки страницы.
-
- 2.  **Page.ts**
- Основной компонент, формирует главную страницу и обрабатывает действия пользователя.
-
- **Ключевые функции и методы:**
- - `loadAndRenderProducts()`: Загружает товары с сервера и отображает их на странице.
- - `EventEmitter`: Система событий для координации различных частей приложения.
-
- **Работа с API:** отправляет запросы к серверу для получения данных о товарах.
-
- **Примечания:** используются утилиты `cloneTemplate`, `ensureElement`, `ensureAllElements` для безопасной работы с DOM.
-
- 3.  **Model.ts**
- Определение структуры моделей данных и методов работы с ними.
-
- **Интерфейсы и классы:**
- - `CartModel`: Класс, реализующий логику корзины (добавление, удаление товаров, расчет итоговой суммы).
-
- **Ключевые методы:**
- - `addToCart()`: добавляет товар в корзину.
- - `removeFromCart()`: удаляет товар из корзины.
- - `clearCart()`: очищает корзину.
- - `totalCount()` и `totalPrice()`: свойства для расчета общего количества и стоимости товаров в корзине.
-
- 4.  **BaseModal.ts**
- Абстрактный класс для создания и управления любыми модальными окнами.
-
- **Основные функции и методы:**
- - `open()`: открывает модальное окно.
- - `close()`: закрывает модальное окно.
- - `replaceContent()`: заменяет содержимое окна новым контентом.
- - `initializeListeners()`: настраивает обработчики событий клавиатуры и мыши.
-
- **Примечания:** используется утилита `bem()` для генерации классов CSS в стиле BEM.
-
- 5.  **Forms.ts**
- Библиотека функций для валидации и обработки форм.
-
- **Ключевые функции:**
- - `validateForm()`: проверка формы заказа на корректность заполнения.
- - `validateContactForm()`: проверка контактных данных.
-
- **Вспомогательные функции:**
- - `isRequired()`: проверка обязательного заполнения поля.
- - `ValidEmail()`: проверка корректности email.
- - `validatePhone()`: проверка корректности телефонного номера.
-
- 6.  **Components.ts**
- Набор компонентов для работы с формами и пользовательским интерфейсом.
-
- **Функции:**
- - `toggleActiveButton()`: переключает активность кнопки.
- - `handleAddressInput()`: обработка изменений адресной строки.
- - `handlePaymentClick()`: обработка выбора способа оплаты.
- - `resetErrors()`: сбрасывает уведомления об ошибках.
- - `displayError()`: выводит сообщение об ошибке рядом с нужным элементом.
-
- **Примечания:** применяются утилиты `setElementData` и `getElementData` для работы с данными элементов DOM.
-
- 7.  **BasketModal.ts**
- Модальное окно корзины.
-
- **Ключевые функции и методы:**
- - `showBasket()`: показывает окно корзины с товарами и ценой.
- - Удаление товаров: удаляет выбранные товары из корзины.
- - Переход к оформлению заказа: возможность перехода к следующему этапу оформления.
-
- **Примечания:** используются утилиты `cloneTemplate`, `ensureElement`, `ensureAllElements` для безопасного взаимодействия с DOM.
-
- 8.  **CardModal.ts**
- Модальное окно детальной карточки товара.
-
- **Основные функции и методы:**
- - `showProductCard()`: показывает подробную информацию о продукте (изображение, название, описание, цену).
- - Добавление в корзину: возможность добавить товар в корзину по нажатию соответствующей кнопки.
-
- **Примечания:** применяется утилита `cloneTemplate` для динамической генерации карточек товаров.
-
- 9.  **OrderModal.ts**
- Модальное окно оформления заказа.
-
- **Ключевые функции и методы:**
- - `showOrder()`: показывает форму заказа с возможностью выбрать способ оплаты и ввести адрес доставки.
- - Проверка формы: проверка правильности введенных данных перед оформлением заказа.
-
- **Примечания:** применяются утилиты `cloneTemplate`, `ensureElement`, `ensureAllElements` для надежной работы с элементами интерфейса.
-
- 10. **ContactsModal.ts**
- Модальное окно ввода контактных данных.
-
- **Основные функции и методы:**
- - `showContacts()`: показывает форму для ввода контактных данных (email, телефон).
- - Отправка данных: обработка и проверка введённых данных перед отправкой.
-
- **Примечания:** используются утилиты `cloneTemplate`, `ensureElement`, `ensureAllElements` для удобной работы с элементами формы.
-
- 11. **SuccessModal.ts**
- Модальное окно успешного завершения покупки.
-
- **Основные функции и методы:**
- - `showSuccess()`: показывает окно с информацией об успешном завершении покупки и общей суммой заказа.
-
- **Примечания:** применяется утилита `cloneTemplate` для быстрой генерации шаблона окна.
